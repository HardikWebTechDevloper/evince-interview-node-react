1. The front end is in React JS, the back end is in Node JS and the Database is in MySQL

2. There are two folders, 1. Front-end 2. Back-end

3. For setup application run below commands.

- In backend

	- `npm i`

	- `npx sequelize db:migarate`

	- `nodemon index.js`

- In front-end,
	
	- `npm i`

	- `npm start`
