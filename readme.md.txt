1. Front-end is in React JS, Back-end is in Node JS and Database is in MySQL

2. There are two folders, 1. Front-end 2. Back-end

3. For setup application run below commands.

- In back-end

	- `npm i`

	- `npx sequelize db:migarate`

	- `nodemon index.js`

- In front-end,
	
	- `npm i`

	- `npm start`
